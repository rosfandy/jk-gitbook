# Editor WYSIWYG

Block-Based Rich Text Editor, Coda Refs.

## Extension

### Block Node

{% embed url="https://www.loom.com/share/3017b3429fa64691b13894cd347ba00d" %}

Block node adalah unit dasar pada editor berbasis blok yang merepresentasikan satu elemen konten mandiri (seperti paragraf, heading, atau tabel) yang dapat disusun, dipindahkan, dan dikelola secara independen dalam struktur dokumen. Node Block terdiri dari:

* **Paragraph**: blok teks dasar untuk menulis isi konten.
* **Heading**: blok judul untuk menandai struktur dan hierarki dokumen.
* **TaskList**: blok daftar tugas yang dapat digunakan untuk checklist pekerjaan.
* **Table**: blok tabel untuk menyusun data dalam bentuk baris dan kolom.
* **Media**: blok untuk menyisipkan gambar, video, atau lampiran.
* **Detail**: blok konten terlipat (_collapsible_) untuk menyimpan informasi tambahan.
* **CodeBlock**: blok untuk menampilkan potongan kode dengan format terstruktur.

{% embed url="https://www.loom.com/share/7a83fe30f5334af49caa3db1b3723116" %}

* **Embed Figma**: elemen untuk menyematkan konten eksternal melalui _figma_

### Inline Text

**Inline text** adalah elemen teks di dalam sebuah block node yang digunakan untuk memberikan format atau atribut khusus pada sebagian teks. Node inline terdiri dari:

* **Bold**: menebalkan teks untuk penekanan.
* **Italic**: memiringkan teks untuk penekanan atau istilah khusus.
* **Underline**: memberi garis bawah pada teks.
* **Link**: menambahkan tautan ke sumber eksternal atau internal.
* **Highlight**: menyorot teks untuk menandai informasi penting.
* **Inline Code**: menampilkan potongan kode singkat di dalam paragraf.
